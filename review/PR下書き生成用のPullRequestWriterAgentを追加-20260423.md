auto-populates PR bodies from recognized template paths such as
`.github/pull_request_template.md` (underscore). With the current name, opening
a PR in GitHub will not preload the template sections introduced here, so
contributors can easily bypass the intended PR format unless they manually
invoke your custom agent.
